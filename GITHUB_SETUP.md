# GitHub Setup for Aurelian Jets

## Create the repository on GitHub

1. Go to **https://github.com/new**
2. **Repository name:** `Aurelian-Jets`
3. **Description:** (optional) Private aviation landing page
4. **Visibility:** Public or Private
5. **Do NOT** add README, .gitignore, or license (we already have these)
6. Click **Create repository**

## Push your code

After creating the repo, run these commands in your project folder:

```bash
git remote add origin https://github.com/YOUR_USERNAME/Aurelian-Jets.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.

If GitHub shows a different URL (e.g. SSH), use that instead of the HTTPS URL above.
